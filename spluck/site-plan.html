Overview

Purpose

Splunk is SIEM(Security Information and Event Management system) that detect, analyze, and response to security incident. Splunk can monitor real time network traffic, but also analyze PCAP files(Stored network traffic data from past). My app, "Spluck" is similar to Splunk, is a lightweight web app that allows users to search and analyze log data stored int JSON format. The user can search, sort, and filter the logs.

Audience

The audience is anyone who is studying cybersecurity, especially, "Splunk!!" For beginners, Splunk is complicated, and it takes time to understand how it works. However, from my app "Spluck," users can get used to basic SPL(search processing language)

Dynamic elements

The site will include a search bar to filter logs in real time, a dynamic log display that updates based on user input, filtering options (by ip address, time, etc). JavaScript will handle DOM manipulation, event listeners, conditional logic, and data processing using arrays and objects.
